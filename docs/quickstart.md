# CodeHelper 快速入门指南

> 5 分钟上手 CodeHelper -- 从安装到完成第一道题

---

## 第一步：了解 CodeHelper

CodeHelper 是一个本地化的编程学习桌面应用，集成以下核心功能：

- **刷题系统** -- 多语言题库，自动判题
- **AI 助手** -- 智能问答，代码解释
- **错题本** -- 自动记录错误，支持复习
- **知识库** -- 本地文档检索（RAG）
- **代码编辑器** -- Monaco Editor，VSCode 同款体验

---

## 第二步：启动应用

1. 运行 CodeHelper 桌面应用
2. 首次启动会自动初始化数据库和同步题库
3. 界面左侧是导航栏，点击图标切换模块

---

## 第三步：完成第一道题

### 选择题目

1. 点击左侧"刷题"图标进入题目列表
2. 从"基础题库"中选择 **"两数之和"**（难度: easy）
3. 查看题目描述和示例输入输出

### 编写代码

在右侧编辑器中编写 Python 代码：

```python
n = int(input())
nums = list(map(int, input().split()))
target = int(input())

hash_map = {}
for i, num in enumerate(nums):
    complement = target - num
    if complement in hash_map:
        print(hash_map[complement], i)
        break
    hash_map[num] = i
```

### 提交与判题

1. 点击 **"提交"** 按钮
2. 系统自动运行所有测试用例
3. 查看判题结果：
   - **通过** -- 所有用例输出与期望一致
   - **未通过** -- 查看哪个用例失败及具体差异

### 查看反馈

- 通过：题目标记为已完成
- 未通过：自动记录到错题本，可查看 AI 分析建议

---

## 第四步：配置 AI 助手

### 添加 AI 配置

1. 点击左侧"设置"图标
2. 在"AI 配置"中添加新配置：
   - **名称**: 自定义名称（如 "DeepSeek"）
   - **API Key**: 你的 API 密钥
   - **Base URL**: API 地址（如 `https://api.deepseek.com`）
   - **模型**: 模型名称（如 `deepseek-chat`）
3. 点击保存，设为默认配置

### 使用 AI 助手

1. 点击左侧"AI 助手"图标
2. 在刷题时，右侧会自动显示 AI 侧边栏
3. 可以：
   - 询问题目解题思路
   - 让 AI 解释你的代码
   - 请求代码优化建议
   - 使用预设提示词快速提问

---

## 第五步：管理错题

1. 点击左侧"错题本"图标
2. 查看所有做错的题目
3. 每道错题显示：
   - 错误次数
   - 错误类型（编译错误、答案错误等）
   - 错误代码与正确代码对比
   - AI 分析
4. 点击"重做"再次练习

---

## 常用操作速查

| 操作         | 方法                               |
| ------------ | ---------------------------------- |
| 切换编程语言 | 题目详情页选择语言标签             |
| 运行代码     | 编辑器中点击"运行"，查看控制台输出 |
| 筛选题目     | 按难度、标签、来源筛选             |
| 搜索题目     | 在列表顶部搜索框输入关键词         |
| 切换主题     | 设置中选择 Catppuccin 配色方案     |
| 导入知识库   | 知识库模块上传 PDF/MD/TXT 文件     |

## 支持的编程语言

| 语言   | 特点                | 适用场景           |

| SQL    | 数据库查询          | 数据分析练习       |
