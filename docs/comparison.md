## vs Cursor

### Cursor 简介

Cursor 是一款 AI-first 的代码编辑器，基于 VSCode 魔改，深度集成 AI 代码生成能力。主打 AI 辅助编码体验，支持代码补全、对话式编程、代码库问答等功能。

### 功能对比

| 维度           | CodeHelper             | Cursor                   |
| :------------- | :--------------------- | :----------------------- |
| **定位**       | 编程学习平台           | AI 编码编辑器            |
| **目标用户**   | 编程学习者、学生       | 专业开发者               |
| **代码编辑器** | Monaco Editor          | VSCode 魔改版            |
| **AI 能力**    | 对话式 AI + 多模型支持 | 代码生成 + 补全 + 库索引 |
| **题库**       | 158+ 内置题目，多来源  | 无                       |
| **判题系统**   | 自动逐用例判题         | 无                       |
| **错题本**     | 自动收集，AI 分析      | 无                       |
| **知识库**     | 本地 RAG 检索          | AI 代码库索引            |
| **价格**       | 免费（MIT 开源）       | $20/月 Pro               |
| **数据存储**   | 本地 SQLite            | 云端                     |
| **离线能力**   | 核心功能可用           | 需联网                   |

### CodeHelper 优势

- **专注学习场景**: 题库、判题、错题本等功能专为编程学习设计
- **完全免费**: MIT 开源，无任何付费门槛
- **数据隐私**: 所有数据存储在本地，不上传云端
- **离线可用**: 核心刷题功能无需联网

### Cursor 优势

- **AI 代码生成更强**: Tab 补全、多行生成、上下文感知更深入
- **代码库级理解**: AI 可以索引整个项目代码库
- **专业开发导向**: 更适合日常工程开发

### 适用建议

- **选择 CodeHelper**: 你是编程学习者，需要刷题、复习、知识管理
- **选择 Cursor**: 你是专业开发者，需要 AI 辅助日常工程编码
- **两者结合**: 用 CodeHelper 刷题学习，用 Cursor 做项目开发

## vs VSCode + Copilot

### VSCode + Copilot 简介

VSCode 是最流行的代码编辑器，GitHub Copilot 是其 AI 代码补全插件。组合使用可以获得 AI 辅助编码体验，但需要分别安装配置，且 Copilot 需要订阅费用。

### 功能对比

| 维度         | CodeHelper      | VSCode + Copilot             |
| :----------- | :-------------- | :--------------------------- |
| **定位**     | 一体化学习平台  | 通用编辑器 + AI 插件         |
| **安装配置** | 一个应用        | 编辑器 + 插件 + 账号配置     |
| **AI 对话**  | 内置，多模型    | Copilot Chat（需订阅）       |
| **代码补全** | Monaco 内置补全 | Copilot 智能补全（更强）     |
| **题库**     | 158+ 内置       | 需安装 LeetCode 插件         |
| **判题**     | 内置自动判题    | 需第三方插件                 |
| **错题本**   | 内置            | 无                           |
| **知识库**   | 内置 RAG        | 需第三方插件                 |
| **价格**     | 免费            | VSCode 免费 + Copilot $10/月 |
| **生态系统** | 专注学习        | 海量插件生态                 |

### CodeHelper 优势

- **零配置开箱即用**: 不需要安装插件、配置账号，开箱即用
- **学习功能内置**: 题库、判题、错题本、知识库全部内置
- **成本为零**: 无需任何订阅费用
- **AI 灵活性**: 支持本地模型，不锁定特定 AI 服务

### VSCode + Copilot 优势

- **插件生态**: 海量插件满足各种需求
- **Copilot 补全**: AI 代码补全能力业界领先
- **语言支持**: 支持几乎所有编程语言
- **社区**: 庞大的用户社区和文档

### 适用建议

- **选择 CodeHelper**: 你主要做编程学习和刷题，希望一个工具搞定
- **选择 VSCode + Copilot**: 你需要全功能开发环境和强大的 AI 补全
- **两者结合**: 用 CodeHelper 刷题学习，用 VSCode 做项目开发

## vs LeetCode

### LeetCode 简介

LeetCode 是全球最流行的在线刷题平台，拥有 2000+ 道题目和庞大的用户社区。提供在线判题、讨论区、竞赛等功能，是面试准备的首选平台。

### 功能对比

| 维度         | CodeHelper                     | LeetCode               |
| :----------- | :----------------------------- | :--------------------- |
| **定位**     | 一体化学习平台                 | 在线刷题平台           |
| **题库规模** | 158+ 精选题目                  | 2000+ 题目             |
| **题目来源** | 力扣、牛客、PAT、CSP、数学建模 | LeetCode 原创          |
| **编辑器**   | Monaco (VSCode 同款)           | 简易在线编辑器         |
| **AI 辅助**  | 内置 AI 对话 + 侧边栏          | 无（LeetCode Discuss） |
| **错题本**   | 自动收集，AI 分析              | 手动标记               |
| **代码运行** | 本地运行，支持六语言           | 在线运行               |
| **离线使用** | 支持                           | 不支持                 |
| **社区**     | 开源社区                       | 庞大用户社区           |
| **价格**     | 免费                           | 免费 / $35/月 Premium  |

### CodeHelper 优势

- **AI 辅助解题**: 刷题时可以直接调用 AI 分析代码
- **多来源题目**: 覆盖力扣、牛客、PAT、CSP 等多个平台的题目
- **自动错题本**: 无需手动标记，自动收集所有失败提交
- **离线可用**: 不依赖网络，随时随地刷题
- **专业编辑器**: Monaco Editor 提供比 LeetCode 更好的编辑体验

### LeetCode 优势

- **题库更大**: 2000+ 题目，覆盖面更广
- **社区活跃**: 讨论区、题解、竞赛系统完善
- **面试认可**: LeetCode 刷题记录被部分公司认可
- **竞赛系统**: 周赛、双周赛等竞赛活动

### 适用建议

- **选择 CodeHelper**: 你希望在本地刷题，需要 AI 辅助和错题管理
- **选择 LeetCode**: 你需要大量题目、竞赛体验和社区讨论
- **两者结合**: 用 CodeHelper 本地练习和错题复习，用 LeetCode 参加竞赛和社区交流

| **定位**     | 编程学习平台       | 通用 AI 助手       |

| **AI 对话**  | 内置，支持多模型   | GPT 系列模型       |
| **题库**     | 158+ 内置          | 无                 |

| **代码运行** | 本地运行六种语言   | Code Interpreter   |
| **数据隐私** | 本地存储           | 云端存储           |
| **价格**     | 免费               | $20/月 Plus        |

### CodeHelper 优势

- **代码编辑内置**: 不需要在编辑器和 ChatGPT 之间切换
- **题库判题一体**: 写代码、判题、问 AI 在同一个窗口完成
- **错题自动管理**: 自动收集错题，AI 分析薄弱点

- **成本为零**: 完全免费，无需订阅

### ChatGPT 优势

- **AI 能力更强**: GPT-4o 等模型能力业界领先
- **通用性**: 不限于编程，可以处理各种任务
- **知识广度**: 训练数据覆盖广泛的知识领域
- **Code Interpreter**: 支持在线运行 Python 代码

### 适用建议

- **选择 CodeHelper**: 你需要完整的编程学习环境，希望编辑器、题库、AI 一体化
- **选择 ChatGPT**: 你需要最强的 AI 对话能力，不局限于编程学习
- **两者结合**: 用 CodeHelper 做日常刷题和学习，用 ChatGPT 处理复杂问题和通用任务

## CodeHelper 的独特优势

### 1. 一体化整合

CodeHelper 是唯一将代码编辑器、AI 对话、题库系统、错题本和知识库整合到一个桌面应用中的编程学习工具。用户无需在多个工具之间切换，所有学习场景在一个窗口内完成。

### 2. 完全开源免费

CodeHelper 基于 MIT License 开源，所有功能完全免费，无付费墙、无订阅、无广告。用户可以自由使用、修改和分发。

### 3. 本地数据隐私

所有数据存储在本地 SQLite 数据库中，包括聊天记录、学习进度、错题数据等。API Key 使用 Electron safeStorage 加密存储。用户数据不上传任何云端服务。

### 4. 灵活的 AI 集成

支持任意 OpenAI 兼容 API，包括商业服务（GPT-4o、Claude 等）和本地模型（Ollama）。用户可以自由选择 AI 服务提供商，不被锁定。

### 5. 离线可用

核心功能（代码编辑、题库刷题、错题本、知识库检索）完全离线可用。只有 AI 对话功能需要网络连接。

### 6. 学习导向设计

每个功能都围绕编程学习场景设计：题库覆盖多个来源和难度级别，错题本自动追踪学习薄弱点，知识库支持课程资料检索，AI 侧边栏实时辅助解题。

## 选择建议

### 你是一个编程初学者

**推荐 CodeHelper**。零成本、零配置，内置题库从基础到进阶，AI 助手随时解答疑问，错题本帮你查漏补缺。

### 你正在准备技术面试

**推荐 CodeHelper + LeetCode**。用 CodeHelper 日常练习和错题复习，用 LeetCode 拓展题目范围和参加竞赛。

### 你是一个专业开发者

**推荐 VSCode + Copilot 或 Cursor**。更强的 AI 代码补全能力，更完善的开发生态。CodeHelper 可作为辅助学习工具。

### 你注重数据隐私

**推荐 CodeHelper**。所有数据本地存储，支持本地 AI 模型，完全不依赖云端服务。

### 你预算有限

**推荐 CodeHelper**。唯一完全免费且功能完整的编程学习平台，无需任何订阅费用。
