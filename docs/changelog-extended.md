# CodeHelper 详细版本历史

本文档是 [CHANGELOG.md](../CHANGELOG.md) 的扩展版，按模块和功能域细分每次变更。格式遵循 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/) 规范。

## [1.0.0] - 2026-06-02

CodeHelper 首个正式版本发布。基于 Electron + React + TypeScript 构建的 AI 驱动桌面编程助手。

---

### Monaco 代码编辑器

| 变更类型 | 说明                                                                      |
| -------- | ------------------------------------------------------------------------- |
| 新增     | 集成 Monaco Editor 0.55（VSCode 同款编辑引擎）                            |
| 新增     | 语法高亮与智能代码补全                                                    |
| 新增     | 多标签页管理，支持同时编辑多个文件                                        |
| 新增     | 代码折叠、括号匹配、自动缩进                                              |
| 新增     | 内置控制台（Console）面板，实时查看运行输出                               |
| 新增     | 标签页状态持久化（内容、光标位置、滚动位置），500ms 防抖写入 localStorage |
| 新增     | Minimap 开关，偏好保存到 localStorage                                     |
| 新增     | 编辑器配置集中管理（`src/utils/monacoConfig.ts`）                         |
| 新增     | 自定义 Monaco 主题（Catppuccin Mocha / Fjord / Ember）                    |
| 新增     | 大文件优化（`largeFileOptimizations: true`）                              |
| 优化     | 编辑器选项缓存为单例对象，避免重复分配                                    |
| 优化     | Monaco worker 预加载与懒加载策略                                          |
