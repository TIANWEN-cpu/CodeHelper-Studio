## Monaco 代码编辑器

### 功能描述

CodeHelper 内置 VSCode 同款 Monaco Editor（v0.55），提供专业级代码编辑体验。支持语法高亮、智能补全、代码折叠、括号匹配、自动缩进等核心编辑功能，配合多标签页管理，可同时编辑多个文件。

### 核心能力

| 能力       | 说明                                                     |
| :--------- | :------------------------------------------------------- |
| 语法高亮   | 支持 Python、C/C++、Java、C#、JavaScript、SQL 等主流语言 |
| 智能补全   | 基于语言服务的上下文感知代码补全                         |
| 多标签页   | 同时打开多个文件，自由切换                               |
| 代码折叠   | 折叠/展开函数、类、注释块                                |
| 括号匹配   | 高亮匹配的括号、花括号、方括号                           |
| 自动缩进   | 根据语言规则自动调整缩进                                 |
| 内置控制台 | 实时查看代码运行输出，无需切换窗口                       |
| 智能粘贴   | 粘贴时自动调整缩进格式                                   |

### 使用场景

- **日常编码**: 在题库系统中编写解题代码
- **代码实验**: 快速测试代码片段和算法逻辑
- **多文件编辑**: 同时查看题目描述和解题代码
- **学习新语言**: 利用语法高亮和补全辅助学习新编程语言

### 用户收益

- 无需安装 VSCode 等外部编辑器，开箱即用
- 与题库系统、AI 对话无缝集成，无需窗口切换
- 编辑器状态与题库、错题本联动，上下文不丢失

## AI 智能对话

### 功能描述

CodeHelper 内置 AI 对话系统，支持任意 OpenAI 兼容 API（如 GPT-4o、本地 Ollama 等）。采用流式输出架构，逐字符渲染响应，提供实时反馈体验。内置预设提示词系统和长期记忆，让 AI 真正成为你的编程助手。

### 核心能力

| 能力          | 说明                               |
| :------------ | :--------------------------------- |
| 流式输出      | SSE 流式传输，逐字符渲染，实时反馈 |
| Markdown 渲染 | 完整支持 Markdown 语法和代码块高亮 |
| 预设提示词    | 内置常用编程提示词，支持自定义保存 |
| 长期记忆      | 跨对话记住用户偏好和编程习惯       |
| 多模型管理    | 支持配置多个 AI 模型，随时切换     |
| API Key 安全  | 使用 Electron safeStorage 加密存储 |

### 使用场景

- **代码解释**: 选中代码片段，请求 AI 逐行解释
- **Bug 排查**: 粘贴错误信息，获取修复建议
- **算法学习**: 请求 AI 讲解算法思路和时间复杂度
- **代码优化**: 让 AI 审查代码并提出优化建议
- **知识问答**: 询问编程概念、语法细节、最佳实践

### 用户收益

- 支持本地 Ollama 模型，离线也能使用 AI 助手
- 长期记忆让 AI 了解你的水平和偏好，回答更有针对性
- 预设提示词系统减少重复输入，提升对话效率

## 题库系统

### 功能描述

CodeHelper 内置 158+ 道编程题目，覆盖力扣、牛客、PAT、CSP、数学建模等多个来源。配备自动判题引擎，支持逐用例运行与精确比较，配合 AI 侧边栏实时辅助解题。

### 核心能力

| 能力       | 说明                                         |
| :--------- | :------------------------------------------- |
| 多来源题库 | 力扣经典 80 题、基础题 48 题、数学建模 30 题 |
| 六语言支持 | Python、C、C++、C#、Java、SQL                |
| 自动判题   | 逐用例运行，精确比较输入输出                 |
| 多维筛选   | 按难度、标签、来源、赛道、平台筛选           |
| AI 侧边栏  | 刷题时直接调用 AI 分析代码                   |
| 提交记录   | 完整保存每次提交的代码和结果                 |

### 使用场景

- **日常刷题**: 按难度和标签系统性练习
- **面试准备**: 集中练习力扣高频题
- **课程作业**: 使用 PAT/CSP 题目配合课程学习
- **算法竞赛**: 数学建模题目训练建模思维
- **SQL 练习**: 专门的 SQL 题目练习数据库查询

### 用户收益

- 一站式刷题，无需在多个平台之间跳转
- AI 侧边栏即时辅助，不需要手动复制代码到 ChatGPT
- 本地判题引擎，无需联网即可刷题
- 多来源覆盖，一个平台满足不同学习需求

## 知识库 RAG 检索

### 功能描述

CodeHelper 提供本地文档检索与知识管理功能。支持导入 PDF、Markdown、TXT 格式文档，系统自动进行文本分块和关键词匹配检索，按相关度排序返回结果。向量嵌入字段已预留，未来可升级为语义检索。

### 核心能力

| 能力       | 说明                           |
| :--------- | :----------------------------- |
| 多格式导入 | 支持 PDF / Markdown / TXT 文档 |
| 大文件支持 | 单文件最大 10MB                |
| 自动分块   | 约 500 字符/块，优化检索精度   |
| 关键词匹配 | 基于关键词的文档检索           |
| 相关度排序 | 按匹配程度排序搜索结果         |
| 向量预留   | 预留向量嵌入字段，支持未来升级 |

### 使用场景

- **课程笔记检索**: 导入课程讲义和笔记，随时检索知识点
- **文档参考**: 导入 API 文档、语言规范等参考资料
- **学习资料管理**: 集中管理散落各处的学习资料
- **论文检索**: 导入学术论文 PDF，快速定位关键段落

### 用户收益

- 本地存储，数据不上传云端，保护隐私
- 即时检索，无需打开外部文件逐页翻找
- 与 AI 对话集成，检索结果可直接用于 AI 分析

## 错题本

### 功能描述

CodeHelper 自动追踪编程练习中的错误，从失败提交中收集错题，记录错误类型和错误次数，支持 AI 分析薄弱知识点，并保存正确代码供一键重做。

### 核心能力

| 能力     | 说明                                 |
| :------- | :----------------------------------- |
| 自动收集 | 从失败提交中自动收集错题             |
| 错误分类 | 编译错误、运行时错误、答案错误、超时 |
| 错误追踪 | 记录每道题的错误次数和历史           |
| AI 分析  | 分析薄弱知识点，给出学习建议         |
| 正确代码 | 记录正确解法，支持对照学习           |
| 一键重做 | 快速重新练习错题                     |

### 使用场景

- **考前复习**: 重点复习高频错题，强化薄弱环节
- **知识查漏**: 通过 AI 分析发现知识盲区
- **进步追踪**: 查看错误次数变化，直观感受进步
- **解法对比**: 对比自己的错误代码和正确解法

### 用户收益

- 自动收集，无需手动记录，不遗漏任何错题
- AI 分析帮助定位知识盲区，针对性学习
- 一键重做降低复习门槛，提升复习效率

## 代码运行器

### 功能描述

CodeHelper 支持六种编程语言的本地代码执行，采用编译与运行阶段分离的架构，实现精确定位编译错误和运行时错误。配备完善的资源限制保护机制，防止代码无限循环或资源耗尽。

### 核心能力

| 能力         | 说明                                   |
| :----------- | :------------------------------------- |
| 六语言支持   | Python、C、C++、C#、JavaScript、SQL    |
| 编译运行分离 | 精确区分编译错误和运行时错误           |
| 内存数据库   | SQL 使用 better-sqlite3 内存数据库执行 |
| 结果格式化   | SQL 查询结果自动格式化为表格           |
| 资源限制     | 10 秒超时、1MB 输出上限、最大 5 并发   |
| 安全执行     | 不使用 shell 模式，防止命令注入        |

### 支持的语言和依赖

| 语言       | 运行方式         | 依赖          |
| :--------- | :--------------- | :------------ |
| Python     | `python` 解释器  | Python >= 3.8 |
| C          | `gcc` 编译后执行 | GCC           |
| C++        | `g++` 编译后执行 | G++           |
| C#         | `dotnet run`     | .NET >= 6     |
| JavaScript | `node` 执行      | Node.js       |
| SQL        | 内存数据库       | 内置          |

### 用户收益

- 无需配置外部编译环境，应用内直接运行代码
- 资源限制保护系统安全，不怕死循环
- 编译运行分离，错误定位更精确

## 安全特性

CodeHelper 在安全性方面采取了多项加固措施，确保用户数据和系统安全。

| 安全措施      | 说明                                                                       |
| :------------ | :------------------------------------------------------------------------- |
| Chromium 沙箱 | `contextIsolation` + `nodeIntegration: false`，渲染进程与 Node.js 完全隔离 |
| API Key 加密  | Electron `safeStorage` API 加密存储敏感配置                                |
| CSP 策略      | 严格 Content-Security-Policy 头部，防止 XSS 攻击                           |
| 资源限制      | 超时控制、输出大小限制、并发数限制                                         |
| IPC 验证      | 所有 IPC 调用进行类型检查与协议白名单校验                                  |
| 无 shell 注入 | 子进程执行不使用 `shell:true`                                              |
| 链接白名单    | 仅允许 `http:` / `https:` 协议的外部链接                                   |
