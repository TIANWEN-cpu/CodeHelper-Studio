# 编辑器指南

CodeHelper 内置 Monaco Editor（与 VS Code 同款），提供专业的代码编辑体验。

## 概览

编辑器模块由以下组件构成：

- **MonacoEditor** - 核心编辑器组件，基于 Monaco Editor 0.55
- **EditorTabs** - 多标签页管理
- **Console** - 内置控制台面板，查看代码运行输出
- **TerminalPanel** - 终端面板集成

## 编辑器功能

### 语法高亮

Monaco Editor 支持主流编程语言的语法高亮，包括但不限于：

| 语言       | 文件扩展名     |
| ---------- | -------------- |
| Python     | `.py`          |
| JavaScript | `.js`、`.jsx`  |
| TypeScript | `.ts`、`.tsx`  |
| C          | `.c`、`.h`     |
| C++        | `.cpp`、`.hpp` |
| Java       | `.java`        |
| C#         | `.cs`          |
| SQL        | `.sql`         |
| Markdown   | `.md`          |
| JSON       | `.json`        |
| HTML       | `.html`        |
| CSS        | `.css`         |

### 智能代码补全

- 基于语言上下文的智能提示
- 代码片段自动展开
- 参数信息提示
- 悬停信息展示

### 代码编辑特性

- **代码折叠** - 折叠/展开代码块（点击行号左侧的折叠箭头）
- **括号匹配** - 自动高亮匹配的括号
- **自动缩进** - 根据语言规则自动调整缩进
- **多光标编辑** - `Alt+Click` 添加多个光标同时编辑
- **块注释切换** - `Ctrl+/` 切换注释
- **代码格式化** - 自动调整代码格式
- **智能粘贴** - 粘贴代码时自动调整缩进

## 多标签页管理

### 标签页操作

| 操作               | 方式                   |
| ------------------ | ---------------------- |
| 新建标签页         | 通过题目详情或手动创建 |
| 切换标签页         | 点击标签页标题         |
| 关闭标签页         | 点击标签页右侧关闭按钮 |
| 重新打开关闭的标签 | 通过题目列表重新打开   |

### 标签页特性

- 每个标签页独立保存代码内容
- 标签页标题显示当前编辑的文件/题目名称
- 未保存修改的标签页会有视觉指示
- 支持同时打开多个标签页并行编辑

## 编辑器设置

在 **设置** 模块中可以自定义编辑器行为：

| 设置项   | 说明                  | 默认值       |
| -------- | --------------------- | ------------ |
| 字体大小 | 编辑器文字大小        | 14px         |
| Tab 宽度 | 每个 Tab 占用的空格数 | 4            |
| 主题     | 编辑器配色方案        | 跟随应用主题 |

## 控制台面板 (Console)

控制台面板位于编辑器下方，用于显示代码运行结果。

### 功能

- **标准输出 (stdout)** - 以绿色显示程序的正常输出
- **标准错误 (stderr)** - 以红色显示错误信息
- **退出码** - 显示程序的退出状态码
- **执行时间** - 显示代码运行耗时
- **编译阶段指示** - 对于 C/C++/C# 等编译型语言，区分编译错误和运行时错误

### 运行代码

1. 在编辑器中编写代码
2. 点击运行按钮或使用快捷键执行
3. 查看控制台面板中的输出结果

### 资源限制

代码运行器内置安全保护机制：

| 限制项     | 值     | 说明               |
| ---------- | ------ | ------------------ |
| 执行超时   | 10 秒  | 超时后自动终止进程 |
| 输出大小   | 1 MB   | 超出限制自动截断   |
| 最大并发数 | 5 个   | 同时运行的进程上限 |
| 代码长度   | 100 KB | 单次提交代码上限   |

## 提交代码到题库

在编辑器中编写的代码可以直接提交到题库系统进行自动判题：

1. 确保当前标签页关联了题目
2. 在控制台面板或题目详情中选择编程语言
3. 点击 **提交** 按钮
4. 系统自动运行所有测试用例并返回结果
5. 判题结果包括：通过 (Accepted)、答案错误、编译错误、运行时错误、超时
