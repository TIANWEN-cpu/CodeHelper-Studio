# CodeHelper Demo Project

本目录包含 CodeHelper 的演示内容，帮助新用户快速了解系统功能。

## 目录结构

```
demo/
  README.md              -- 本文件
  project-structure.md   -- 示例项目结构说明
  solutions/             -- 多语言示例题解
    python/              -- Python 题解
    cpp/                 -- C++ 题解
    c/                   -- C 题解
    csharp/              -- C# 题解
    sql/                 -- SQL 题解
```

## 使用方法

1. 在 CodeHelper 中选择"刷题"模块
2. 从题库中选择一道题目
3. 阅读题目描述和示例
4. 在编辑器中编写代码，可参考 `solutions/` 中的题解
5. 点击"提交"查看判题结果

## 题目难度分布

| 难度   | 数量 | 说明                           |
| ------ | ---- | ------------------------------ |
| easy   | ~20  | 入门基础题，适合初学者         |
| medium | ~12  | 中等难度，需要一定算法基础     |
| hard   | ~5   | 高难度，涉及复杂算法与数据结构 |

## 支持的编程语言

- **Python** - 适合快速原型和算法竞赛
- **C** - 适合系统编程和指针练习
- **C++** - 适合算法竞赛和 STL 学习
- **C#** - 适合 .NET 开发者
- **SQL** - 适合数据库查询练习
