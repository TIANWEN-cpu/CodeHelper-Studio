# 苏格拉底式复盘：让 AI 少讲答案

当你想真正学会一个知识点，最好让 AI 少讲完整答案，多问你判断题和追问。

## 复盘 Prompt

```text
请不要直接讲答案。
围绕我刚做错的题，用苏格拉底式提问帮助我复盘。
每次只问一个问题，等我回答后再继续。
最后请帮我整理：错因、正确思路、可迁移模板、下一道练习。
```

## 错题转知识卡

| 字段     | 写法                     |
| -------- | ------------------------ |
| 错因     | 我误判了什么条件         |
| 触发信号 | 下次看到什么关键词要警觉 |
| 正确模板 | 2-5 行伪代码或步骤       |
| 反例     | 一个能打破错误思路的例子 |

> 复盘不是重新看一遍答案，而是让自己能在新题里识别同一种结构。
