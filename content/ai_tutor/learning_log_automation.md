# 自动化学习日志：让 AI 帮你复利

学习日志不是流水账。它应该把今天做了什么、卡在哪里、下一步做什么，用稳定格式沉淀下来，方便 AI Tutor 下次接上上下文。

## 日志字段

| 字段   | 说明                         |
| ------ | ---------------------------- |
| 主题   | 今天学习或练习的核心内容     |
| 完成   | 已经做完的课程、题目或代码   |
| 卡点   | 仍然不确定的概念、错误或决策 |
| 下一步 | 明天最小可执行动作           |

## Markdown 模板

```md
## 主题

## 完成

## 卡点

## 下一步
```

## 自动化流程

1. 从课程、题库、错题和 AI 对话中收集事实。
2. 让 AI 只总结事实，不夸大进度。
3. 把卡点转换成下一次学习的输入。
4. 每天保留一条最小行动。

> 好日志的标准是：明天打开它，你知道第一件事该做什么。

## 小练习

实现 `build_learning_log(topic, done, stuck, next_step)`，输出包含四个二级标题的 Markdown 学习日志。
